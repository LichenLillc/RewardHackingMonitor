# Reward Hacking Monitor Project Page

Static project website for:

**Do Prompt-Elicited Trajectories Reflect Training-Time Reward Hacking? A Systematic Study on Monitoring Trainig-Time Reward Hacking in Code Generation**

The site is built from the Academic Project Page Template, with layout inspiration from the Self-Evolving Visual Questioner project page.

## Structure

- `index.html`: project page content and metadata
- `static/css/index.css`: custom visual styling
- `static/js/index.js`: scroll-to-top and BibTeX copy behavior
- `static/images/`: rendered figures used by the page
- `static/pdfs/rhmonitor-paper.pdf`: paper PDF

## Local Preview

Open `index.html` directly in a browser, or run a local static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/`.
